# 首页轮播图管理简化方案

**版本**: v1.0  
**日期**: 2026-06-11  
**目标**: 简单易用的首页轮播图管理

---

## 功能概述

**简单明了**：只管理首页轮播图，支持：
- 纯轮播图（只有图片）
- 轮播图+标题（有标题）  
- 轮播图+标题+描述（完整信息）

**管理操作**：
- ✅ 上传轮播图（拖拽或粘贴 URL）
- ✅ 添加可选标题和描述
- ✅ 设置跳转（分类/H5 链接/小程序）
- ✅ 启用/禁用
- ✅ 编辑和删除

---

## 数据模型

### 数据库表修改

只需在 `ads` 表增加一个字段：

```sql
ALTER TABLE `ads` ADD COLUMN IF NOT EXISTS `description` TEXT COMMENT '轮播图描述';
```

**字段说明**：
| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | VARCHAR(36) | 轮播图 ID |
| `position_id` | VARCHAR(36) | 广告位 ID（首页轮播图） |
| `title` | VARCHAR(128) | 标题（可选） |
| `description` | TEXT | 描述（可选） |
| `image` | VARCHAR(255) | 轮播图 URL |
| `link_type` | VARCHAR(32) | 跳转类型（category/h5/miniapp） |
| `link_value` | VARCHAR(128) | 跳转值（分类code/URL/路径） |
| `status` | VARCHAR(32) | 启用/禁用 |
| `created_at` | TIMESTAMP | 创建时间 |
| `updated_at` | TIMESTAMP | 更新时间 |

---

## 后端 API

**现有 API**（无需改动）：
```
GET    /api/admin/ads/positions           - 获取广告位及其轮播图
POST   /api/admin/ads                     - 创建轮播图
PUT    /api/admin/ads/:id                 - 更新轮播图
DELETE /api/admin/ads/:id                 - 删除轮播图
```

只需确保：
1. `POST /api/admin/ads` 接受 `title` 和 `description` 字段
2. `PUT /api/admin/ads/:id` 支持更新这两个字段

async function saveAd() {
