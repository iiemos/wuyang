青柠清新风APP UI 前端开发设计规范文档

一、设计整体概述

本套UI为年轻化轻治愈·青柠清新液态风格，主打浅绿渐变活力调性，适配生活服务、出行、电商、育儿、回收、社交副业等全品类大众向APP，视觉柔和透气、交互轻快活泼，适配移动端H5/小程序/APP原生开发，可直接供给AI编码精准还原落地。
二、主题色彩规范

1. 主色调（品牌核心色）
色彩用途 色值代码 应用场景 
主色（亮青柠绿） #79F89D 主要按钮、选中态、主图标、底部导航高亮、核心角标 
主色渐变 起点#B1FCCC → 终点#79F89D 页面头部背景、卡片渐变底色、大区块背景 

2. 辅助色
色彩名称 色值代码 应用场景 
浅薄荷绿 #BEFDF7 次要卡片、未选中态、辅助模块底色 
浅紫辅助色 #CBCDFE / #9B9EFC 标签、装饰渐变、区分次要功能模块 
警示橙/强调色 #FF9F29 优惠、价格、活动标签、强提醒 
成功色 #34D19D 完成、达标、收益提示 
警告色 #FFB800 任务提醒、等级升级 

3. 中性色（文字/背景）
层级 色值代码 应用场景 
一级文字（标题） #1A1A1A 页面大标题、卡片主文案 
二级文字（常规正文） #333333 列表正文、常规说明文字 
三级文字（辅助信息） #666666 时间、距离、次要备注、小字描述 
置灰文字（失效） #999999 未解锁、禁用状态文字 
页面底色 #FFFFFF / #F8FCF9 全局页面基础背景 
分隔线/描边 #E8F3ED 卡片分割、模块边框、分割线 

4. 禁用/透明态

• 禁用按钮：主色透明度 40%，文字色 #AAAAAA

• 遮罩底色：纯黑 #000000，透明度 60%
三、字体排版规范（移动端基准 375px 设计稿）

1. 字体全局规则

• 全局字体：PingFang SC、苹方、思源黑体、system-ui

• 字体行高：字号+8px，保证字间距舒展透气

• 字重区分：

◦ 标题：600/700（加粗）

◦ 正文常规：400（常规）

◦ 强调文字：500（中粗）

2. 字号尺寸对照表
文字层级 字号大小 字重 使用位置 
超大标题 28px 700 页面顶部大标题、活动主标题 
一级标题 20px 600 卡片标题、模块分区标题 
二级标题 18px 500 列表主标题、弹窗标题 
正文标准 16px 400 页面常规正文、按钮文字 
辅助小字 14px 400 描述文案、列表副标题 
备注极小字 12px 400 标签、时间、角标、底部备注 

四、间距与边距栅格规范

1. 基础栅格单位

基准间距单位：8px，所有间距为8的整数倍

2. 全局边距

• 页面安全左右边距：16px（所有内容距离屏幕左右预留16px，不贴边）

• 顶部状态栏安全边距：44px

• 底部导航安全边距：34px

3. 模块间距
间距用途 间距数值 
卡片内部内边距 16px 
卡片与卡片上下间距 12px / 16px 
模块大分区间距 24px / 32px 
文字行段落间距 8px 
按钮内部上下内边距 10px 上下 / 16px 左右 
图标与文字间距 6px / 8px 

五、按钮组件详细规范

1. 主要主按钮（全局CTA操作、立即购买、下一步、确认）

• 背景色：主色渐变 #B1FCCC → #79F89D

• 文字颜色：纯白色 #FFFFFF

• 文字字号：16px，字重 600

• 圆角：16px（大圆角柔和风格）

• 高度：48px

• 内边距：左右 24px，上下 12px

• 描边：无

• 阴影：0px 4px 12px rgba(121, 248, 157, 0.35)

• 点击态：整体缩小缩放 0.96，透明度轻微降低

• 禁用态：背景色透明度40%，无阴影，文字#AAAAAA

2. 次要次按钮（查看详情、了解更多、加入购物车）

• 背景色：纯白色 #FFFFFF

• 文字颜色：一级文字 #1A1A1A

• 边框：1px 实线 #D1F6DD

• 圆角：16px

• 高度：44px

• 字号：15px，字重 500

3. 深色辅助按钮（分享购买、取消、次要操作）

• 背景色：纯黑色 #222222

• 文字颜色：纯白色

• 圆角：16px

• 高度：44px

• 字号：15px

4. 小型标签按钮（福利、券、筛选标签）

• 背景色：亮绿色/浅黄色主题色

• 圆角：12px

• 高度：32px

• 字号：12px

• 左右内边距：12px

5. 文字按钮（纯文字跳转）

• 仅文字，无边框无底色

• 文字色：主色 #79F89D

• 字号：14px

• 点击：文字轻微放大变色加深

6. 图标按钮

• 尺寸：40px×40px

• 圆角：12px

• 背景：浅薄荷绿底色

• 图标大小：20px居中
六、图标Icon风格规范

1. 整体风格：轻量化线性+面性结合、圆润可爱、微Q版年轻化

2. 线条粗细：2px统一描边

3. 圆角：图标边角统一圆角 4px，拒绝尖锐直角

4. 主色图标：主色青柠绿填充

5. 普通功能图标：中性深灰 #333333

6. 未选中态：浅灰色 #999999

7. 尺寸规范：

◦ 底部tab图标：24px×24px

◦ 功能入口图标：20px×20px

◦ 列表内小图标：16px×16px

8. 附加细节：少量图标搭配卡通可爱小元素、轻微外发光点缀，增强趣味性
七、卡片组件设计规范

1. 全局卡片圆角：20px

2. 卡片底色：纯白色

3. 卡片阴影：0 2px 10px rgba(0,0,0,0.06)

4. 卡片边框：极浅1px #F0F7F2 可选

5. 卡片内边距：16px

6. 特殊渐变卡片：顶部1/3区域使用主色渐变打底，下半部分纯白过渡

7. 列表卡片上下间距：12px
八、弹窗&浮层组件规范

1. 居中全局弹窗

• 弹出方式：底部向上平滑滑入，缓动300ms

• 弹窗圆角：顶部24px，底部24px

• 最大宽度：屏幕宽度-48px

• 背景：纯白色

• 遮罩：全屏黑色半透明，背景模糊轻微虚化

• 底部操作按钮：双按钮左右均分，主按钮+次按钮组合

• 关闭交互：点击遮罩可关闭、右上角关闭icon

2. 底部抽屉弹窗（半屏弹窗）

• 弹出方式：从屏幕底部向上弹性滑出，超出回弹阻尼效果

• 圆角：顶部圆角 24px，底部直角

• 高度：占据屏幕60%-75%高度

• 遮罩跟随渐变透明度变化

3. 局部功能浮层（红包、任务、活动提示、优惠券弹窗）

• 弹出方式：中心缩放弹出，先放大再回弹（scale 0→1.05→1）

• 圆角：20px

• 底色：主色浅渐变+白色搭配

• 风格：搭配卡通IP形象、可爱装饰元素，轻量化轻量化悬浮感

4. 全局Toast轻提示

• 位置：屏幕底部居中

• 圆角：12px

• 背景色：rgba(0,0,0,0.75)

• 文字：白色14px

• 时长：2s自动消失
九、导航与底部Tab栏规范

1. 底部导航栏高度：56px + 底部安全区

2. Tab圆角：顶部圆角16px

3. 未选中态：图标+文字 浅灰色 #999999

4. 选中态：图标+文字 主色青柠绿，图标轻微放大上浮

5. 顶部导航栏高度：48px

6. 导航文字：18px 600字重，居中对齐
十、全局交互与动效效果规范

1. 基础动效时长

• 所有过渡动画：标准300ms 缓动ease-out

• 弹性动效：350ms

• 无生硬线性动画

2. 页面切换交互

• 页面左右跳转：左右平滑侧滑入场出场

• 上下页面：垂直淡入+位移滑动

3. 点击反馈交互

• 所有可点击元素：按下时缩放scale 0.96，松开回弹

• 按钮 hover/按压：阴影加深、亮度轻微提升

• 列表item点击：轻微上浮位移+底色变暗

4. 加载与空状态

• 加载动画：主色绿色旋转渐变加载、流畅循环

• 空状态：搭配简约可爱卡通插画+柔和浅绿背景+引导按钮

5. 特殊趣味交互

• 活动弹窗、升级弹窗：带轻微弹跳回弹效果

• 数字变动（收益、金额）：数字滚动递增动画

• 点赞、完成操作：绿色粒子飘散轻动效
十一、页面通用布局规则

1. 全局采用流式居中布局，所有内容居中对齐

2. 信息层级从上到下：大Banner入口→功能入口宫格→核心内容列表→底部推荐

3. 宫格入口布局：每行3-4个icon，均匀等分，间距12px

4. 列表布局：左头像+中间文字信息+右侧操作按钮，信息对齐规整
AI编码读取关键提取速览（核心一键落地）

1. 主色渐变：#B1FCCC ~ #79F89D，全局清新绿

2. 主要按钮：圆角16px、48px高、16px白字、绿色渐变、柔和外阴影

3. 全局圆角统一：卡片20px、按钮16px、小标签12px

4. 基准字号：标题20px、正文16px、辅助14px、小字12px

5. 全局安全边距左右16px，栅格间距8px倍数

6. 图标圆润轻量化，主绿次灰

7. 弹窗底部上滑、居中弹窗缩放弹出，300ms缓动

8. 点击全元素scale 0.96按压反馈，整体轻快年轻化
